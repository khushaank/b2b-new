Online Continuous Emission Monitoring Systems

OCEMS is a mandatory requirement for industries as per CPCB (Central Pollution Control Board) and SPCB (State Pollution Control Boards) regulations. B2B Industrial Solutions provides end-to-end OCEMS solutions including system design, installation, calibration, and connectivity to regulatory servers.

Our systems continuously monitor stack emissions and transmit real-time data to pollution control boards, ensuring regulatory compliance and avoiding penalties.

Parameters Monitored

Particulate Matter

SPM (Suspended Particulate Matter) using opacity meters or laser-based analyzers.

Gaseous Pollutants

SO₂, NOₓ, CO, CO₂, HCl, and other gases using extractive or in-situ analyzers.

Stack Parameters

Flow rate, temperature, pressure, and oxygen concentration for emission calculations.

System Components

  • Gas Analyzers: NDIR, chemiluminescence, electrochemical, and FTIR analyzers for multi-gas detection.
  • Dust Monitors: Opacity meters, light scattering devices, and beta attenuation monitors for particulate measurement.
  • Sample Conditioning: Heated probes, filters, moisture traps, and chillers to prepare samples for analysis.
  • Data Acquisition: PLC/DCS-based control systems with local display and data logging.
  • Server Connectivity: GSM/GPRS/4G modems for real-time data transmission to CPCB/SPCB servers.
  • Calibration System: Certified calibration gases and automated calibration routines.

Our Services

  • Turnkey Installation: Complete OCEMS supply, installation, commissioning, and regulatory approvals.
  • CPCB/SPCB Compliance: System configuration as per latest pollution control board guidelines.
  • Server Integration: Setup and testing of data connectivity to state/central servers.
  • Annual Maintenance: Preventive maintenance, calibration, and spare parts supply.
  • Training: Operator training and documentation for in-house staff.
